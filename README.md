Logistic Map Simulation (Chaos Theory)

This project demonstrates the behavior of the logistic map, a simple nonlinear equation that exhibits chaotic dynamics.

Equation
xₙ₊₁ = r xₙ (1 − xₙ)

Description
The logistic map models population growth. For certain values of r, the system becomes chaotic.

What I Did
- Implemented the logistic map using Python
- Simulated system behavior over iterations
- Visualized transition from stability to chaos

Tools Used
- Python
- NumPy
- Matplotlib
